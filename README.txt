En este repositorio hay dos proyectos:

   APIRest     - Cliente REST para la consulta de datos en formato JSON. 
                 Admite dos peticiones: Consulta de empresas disponibles y Consulta de datos de contacto de una empresa.	     	      
   APIRestTest - Conjunto de pruebas en JUNIT3

La aplicación se ha desarrollado con el IDE Eclipse sobre plataforma Android 4.0.3.

La documentación de ambos se encuentra en sus respectivas carpetas "\doc".
